# game-of-life

This java program implements [Conwayâ€™s Game Of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

## Usage

Compilation:

```bash
javac *.java
```
Run:

```bash
java RunGameOfLife
```
